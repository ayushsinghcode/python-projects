# 🧮 Python Calculator

A simple command-line calculator built using Python. This project performs basic arithmetic operations and allows users to perform multiple calculations using a loop.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔄 Continuous calculations using a loop
- ⚠️ Handles division by zero

## 📸 Preview

```text
Enter a number: 10
Enter another number: 5
Enter operation (+, -, *, /): *

The result of 10 * 5 is: 50
```
